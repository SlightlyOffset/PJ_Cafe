# Java 2D Horror Sidescroller Prototype

A narrative-driven 2D puzzle-platformer that blends classic sidescrolling with "meta-horror" psychological thriller elements. This project utilizes unique "window-hopping" mechanics and OS-level interactions to create a tense, fourth-wall-breaking experience.

## 👁️ Project Vision
The game challenges players to solve puzzles that exist both within the game world and across their desktop environment. By dynamically moving, resizing, and spawning independent windows, the game creates a psychological thriller atmosphere inspired by titles like *Rhythm Doctor* and modern meta-horror games.

## 🛠️ Tech Stack
- **Language:** Java 17+
- **Window Management:** Java AWT/Swing
- **Game Engine:** LibGDX
- **Build System:** Gradle
