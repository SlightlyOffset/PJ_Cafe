# Product Definition - Java 2D Horror Sidescroller

## Vision
A narrative-driven Java 2D puzzle-platformer that blends classic sidescrolling with "meta-horror" psychological thriller elements. The game uses "window-hopping," desktop popups, and unsettling visual effects to break the fourth wall, challenging the player to solve puzzles that exist both within the game world and across their OS environment.

## Target Audience
- **Fellow students and professors:** To demonstrate mastery of OOP principles, specifically in managing complex state transitions, multi-window synchronization, and advanced visual rendering techniques.

## Core Goals
- **Narrative Depth:** Deliver a compelling psychological thriller story through environmental storytelling and meta-interactions.
- **Technical Innovation:** Implement a "window-hopping" system and multi-window coordination as core puzzle-solving mechanics.
- **Atmospheric Immersion:** Use dynamic visual effects (vignette, screen shake) and OS-level manipulation to maintain constant tension.

## Key Features
- **Window-Hopping Gimmick:** The game viewport dynamically detaches, resizes, and moves across the desktop, serving as both a source of tension and a puzzle mechanic.
- **Psychological Popups:** Independent windows (e.g., watching eyes, cryptic clues) that appear on the desktop to interact with the player or provide puzzle hints.
- **Atmospheric Visuals:** Implementation of pixel art enhanced by procedural effects like vignette (for tunnel vision/stress) and screen shake (for impact and psychological cues).
- **Story-Driven Puzzles:** Puzzles that require the player to interpret the narrative and potentially interact with the windows/popups themselves to progress.

## Gameplay & Challenges
- **Puzzle-Platforming:** Navigating traditional obstacles while managing the "meta" challenge of the moving game window.
- **Psychological Stressors:** Using vignette effects to limit visibility during high-tension moments and screen shake to disorient the player.
- **Environmental Storytelling:** Finding clues within the shifting windows and popups to solve the mystery and advance the plot.
