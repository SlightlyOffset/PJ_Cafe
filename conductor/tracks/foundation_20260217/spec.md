# Track Specification: Project Foundation and Core Meta-Mechanics

## Overview
This track focuses on setting up the technical foundation for the Java 2D Horror Sidescroller. It includes the project scaffolding, the Window Manager (for window-hopping), and a basic platformer controller.

## Requirements
- **Project Scaffold:** Gradle-based Java 17 project with LibGDX and AWT/Swing integration.
- **Window Manager:** A module capable of programmatically moving and resizing the main game window on the desktop.
- **Core Controller:** A simple character controller with gravity, horizontal movement, and jumping.
- **Basic Collision:** A simple ground plane to demonstrate movement within a shifting window.

## Success Criteria
- The project builds and runs via Gradle.
- A game window appears and can be moved programmatically (e.g., via a test trigger).
- A player character can move and jump on a stationary or moving ground platform.
